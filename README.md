# 🚀 Fawad Perzad – Portfolio

**Live Demo:** [fawadperzad.github.io/fawad-perzad-portfolio](https://fawadperzad.github.io/fawad-perzad-portfolio)

![Deploy Status](https://github.com/fawadperzad/fawad-perzad-portfolio/actions/workflows/deploy.yml/badge.svg)

A modern, responsive personal portfolio built with **React**, **TypeScript**, and **Vite**, deployed automatically via **GitHub Actions** to **GitHub Pages**.

## ✨ Features

- ⚡ Lightning-fast performance with Vite
- 🎨 Smooth animations and scroll effects
- 📱 Fully responsive design
- 🧩 Modular component architecture
- 🚀 CI/CD pipeline with automatic deployment
- 📦 Clean, readable codebase

## 🛠️ Tech Stack

| Category   | Technology                          |
| ---------- | ----------------------------------- |
| Frontend   | React 18, TypeScript                |
| Build Tool | Vite                                |
| Styling    | Tailwind CSS / CSS Modules (adjust) |
| Deployment | GitHub Pages + GitHub Actions       |
| Linting    | ESLint (if configured)              |

## 🚀 Getting Started (Run Locally)

```bash
# Clone the repository
git clone https://github.com/fawadperzad/fawad-perzad-portfolio.git

# Navigate into the project
cd fawad-perzad-portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```
